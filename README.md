# Assessment Data

Four CSV files for the technical assessment.

## Files

| File | Description |
|------|-------------|
| `quarterly_financials.csv` | Quarterly financials for 25 US retail companies (Q1 2022 – Q4 2024) |
| `daily_spreads.csv` | Daily bond spreads with bid/ask (Jan 2023 – Dec 2024) |
| `bond_reference.csv` | Bond details: coupon, maturity, size, liquidity tier |
| `credit_events.csv` | Notable credit events (rating changes, earnings, covenant issues) |
